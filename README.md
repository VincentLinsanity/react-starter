# Pre Requirement
- Nodejs
- Gulp

# Quick Start by Default
git clone project
```bash
$ git clone https://github.com/VincentLinsanity/react-starter.git
```

initial node_module
```bash
$ npm install
```

run server
```bash
$ npm install gulp -g
$ gulp
```
